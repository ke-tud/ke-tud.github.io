BlueJ

Als Entwicklungsumgebung für Java werden wir in der Allgemeinen Informatik 2 BlueJ in der Version 2.13 benutzen.

Die Entwicklungsumgebung ist auf den Poolrechnern installiert, zum Starten einfach bluej eingeben.
Einen Download-Link zu BlueJ finden Sie im Abschnitt Wichtige / interessante Links

Wichtige / interessante Links

  • Die offizielle Schnittstellen-Spezifikation von Java SE 6 / Java-API:
    http://java.sun.com/javase/6/docs/api/

  • Download des Java-SDK (Java-Compiler) "JDK 6.0":
    http://java.sun.com/javase/downloads/index.jsp

  • BlueJ 2.13:
    http://www.bluej.org/download/download.html

  • BlueJ-Installationsanleitung:
    http://www.bluej.org/download/install.html

  • BlueJ-Tutorial (PDF-Datei):

  • Die Java-Tutorials von Sun:
